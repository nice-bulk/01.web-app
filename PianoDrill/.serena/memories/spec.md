# PianoDrill - 仕様書

## アプリ概要
**PianoDrill** は、音符やキー（調）を瞬時に当てられるようになるための練習用Webアプリ。

---

## モード一覧

### 1. Note Mode（音符あてモード）
- **目的**: 譜面と鍵盤の位置を直結させる
- **出題**: 五線譜 + 調号 + 音符を表示
  - 単音・和音の両方に対応（設定で切り替え）
  - 表示する譜表: トレブル譜（ト音記号）・バス譜（ヘ音記号）両対応（設定で切り替え）
  - 音域: トレブル譜 C4〜G5、バス譜 E2〜C4（加線は最小限）
- **回答**: 画面上の仮想鍵盤をクリックして回答
  - 単音モード: 1鍵クリックで即時判定
  - 和音モード: 複数鍵を選択し「確定」ボタンを押して判定
- **出題順**: ランダム
- **正誤判定**: 正解なら次の問題へ。不正解はフィードバック表示

### 2. Key Mode（キーあてモード）
- **目的**: 調号を見て何長調（短調）かを瞬時に判別する理論スキルを鍛える
- **出題**: 調号（♯・♭の塊）のみを五線譜上に表示。音符は出さない
- **回答**: ボタン選択（例：「G Major」「E Minor」など）
- **出題順**: ランダム
- **出題範囲**:
  - フェーズ1（現在）: 長調のみ（C〜Cb、全15調）
  - フェーズ2（将来）: 短調（平行調）も追加
- **正誤判定**: 選択即時判定。正解なら次の問題へ。不正解はフィードバック表示

---

## 共通仕様

### 対応環境
- PCブラウザメイン（Chrome / Firefox / Safari）
- スマホ・タブレットはサブ対応

### スコア・履歴
- フェーズ1では不要（シンプルな練習のみ）
- 将来的に正解率・タイムを記録する機能を追加予定

### 将来対応
- MIDIキーボード入力（WebMIDI API）
- 短調（平行調）対応
