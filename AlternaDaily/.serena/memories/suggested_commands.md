# Suggested Commands (Windows / pnpm)

## Development
```
pnpm dev          # Start dev server (Vite HMR)
pnpm preview      # Preview production build locally
```

## Build
```
pnpm build        # TypeScript type-check + Vite production build
```

## Linting
```
pnpm lint         # Run ESLint on all TS/TSX files
```

## Package Management
```
pnpm install      # Install dependencies
pnpm add <pkg>    # Add a package
pnpm add -D <pkg> # Add a dev dependency
```

## System Utils (Windows)
```
git               # Version control
dir               # List directory contents (Windows)
cd                # Change directory
findstr           # Search pattern in files (like grep)
copy / xcopy      # File copy
```
