# AlternaDaily — 一日一変

> 毎日ひとつ、新しい何かを体験する。小さな変化が、豊かな人生をつくります。

## コンセプト

**「一日一変」** をテーマにした PC 向けデイリーミッションアプリです。  
あなたの趣味趣向と難易度設定をもとに、Gemini AI が毎日ひとつのミッションを提案します。  
やった日もやれなかった日も、すべてを記録して積み重ねていきます。

---

## デモモード

**APIキーなしでもすぐに試せます。**

`VITE_GEMINI_API_KEY` が未設定の場合、自動的に**デモモード**で動作します。

- モックデータでミッションが生成されます
- サイドバーに `DEMO` バッジが表示されます
- ホーム画面にデモモード中である旨の通知が表示されます
- それ以外の全機能（履歴・ストリーク・設定など）は通常通り動作します

---

## 画面構成

左サイドバーナビ ＋ メインコンテンツの2カラムレイアウトです。

| 画面 | 内容 |
|---|---|
| 🏠 ホーム | 今日のミッション表示・ガチャ・達成記録。右に難易度サマリーと興味タグを表示 |
| 📅 履歴 | 過去のミッション一覧（グリッド表示）・総数・達成数・達成率 |
| ⚙️ 設定 | ニックネーム・興味・難易度の編集。右にリアルタイムプレビューを表示 |

---

## 機能

- 🎲 **デイリーミッションガチャ** — Gemini API があなた専用のミッションを動的生成
- 🔄 **引き直し機能** — 1日1回だけ別のミッションに引き直せる（引き直し後は必ずそれが今日のミッション）
- ✅ **シンプルな達成記録** — 「達成」「未達成」を記録するだけ。スキップという概念なし
- 🔥 **ストリーク表示** — 連続達成日数をサイドバーとホームに表示
- 📅 **履歴画面** — 過去のミッションと達成率をグリッドで一覧表示
- ⚙️ **難易度設定（3軸）** — 時間コスト・移動コスト・心理的ハードルをそれぞれ設定

---

## ミッションカテゴリ

| カテゴリ | 例 |
|---|---|
| 🏃 行動系 | いつもと違う道で帰る、行ったことない店に入る |
| ✨ 体験系 | 知らない料理を作る、新しいジャンルの音楽を聴く |
| 💭 思考系 | 自分と反対の意見を調べる、知らない言葉を調べる |
| 🤝 人間関係系 | 久しぶりの人に連絡する |

---

## 難易度の3軸

| 軸 | 選択肢 |
|---|---|
| 🕐 時間コスト | 5分程度 / 30分程度 / 半日程度 |
| 🚶 移動コスト | 家でできる / 近所で完結 / どこでもOK |
| 💬 心理的ハードル | 一人でできる / 少し勇気がいる / 知らない人が関わる |

## セットアップ

### 1. 依存パッケージのインストール

```bash
pnpm install
```

### 2. Gemini APIキーの設定（任意）

```bash
copy .env.example .env
```

`.env` を開き、`VITE_GEMINI_API_KEY` に [Google AI Studio](https://aistudio.google.com/) で取得した APIキーを設定してください。

```env
VITE_GEMINI_API_KEY=your_gemini_api_key_here
```

> キーを設定しない場合はデモモードで動作します。

### 3. 開発サーバー起動

```bash
pnpm dev
```

ブラウザで `http://localhost:5173` を開いてください。

## Vercel へのデプロイ

1. このリポジトリを GitHub に push する
2. [Vercel](https://vercel.com/) でプロジェクトをインポートする
3. 環境変数 `VITE_GEMINI_API_KEY` を設定する（任意・未設定ならデモモードで公開）
4. デプロイする

## データ設計

すべてのデータはブラウザの `localStorage` に保存されます。サーバーやログインは不要です。

```
alternadaily_data
├── profile
│   ├── nickname           ニックネーム
│   ├── interests[]        趣味・興味タグ
│   └── difficulty         難易度設定（3軸）
├── records[]
│   ├── mission            ミッション内容
│   ├── status             completed / failed / pending
│   └── completedAt        達成日時（達成時のみ）
└── todayMission           今日のミッション（引き直しフラグ含む）
```
