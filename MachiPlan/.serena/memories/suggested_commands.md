# 開発コマンド（MachiPlan）

## パッケージマネージャー
pnpm を使用

## よく使うコマンド（Windows / PowerShell）

| コマンド | 説明 |
|---|---|
| `pnpm dev` | 開発サーバー起動（Vite HMR） |
| `pnpm build` | 本番ビルド（tsc -b && vite build） |
| `pnpm lint` | ESLint によるコードチェック |
| `pnpm preview` | ビルド成果物のプレビュー |

## ファイル操作（Windows）
- ディレクトリ一覧: `dir` または `ls` (PowerShell)
- ファイル検索: `Get-ChildItem -Recurse`
- Git: 通常の `git` コマンドが使用可能
