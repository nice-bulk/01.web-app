# MachiPlan - Project Overview

## 目的
MachiPlan は React + TypeScript + Vite を使ったフロントエンドWebアプリケーション。
現在は初期テンプレート状態（Vite + React のデフォルト構成）で、アプリ本体はこれから開発予定。

## テックスタック
- **フレームワーク**: React 19
- **言語**: TypeScript 5.9
- **ビルドツール**: Vite 7
- **パッケージマネージャー**: pnpm
- **リンター**: ESLint 9 + typescript-eslint + eslint-plugin-react-hooks + eslint-plugin-react-refresh

## プロジェクト構造
```
MachiPlan/
├── src/
│   ├── App.tsx         # メインコンポーネント

│   └── assets/

├── package.json
├── vite.config.ts
├── tsconfig.json

```
