# キモチミュージック 🎵

日本語の感情タグを選ぶと、その気分にぴったりの邦楽・アニメOSTを1曲提案するWebアプリ。

**「なんか泣きたい」「静かに燃えてる」** — Spotifyのざっくりした気分カテゴリでは拾えない、日本語特有の細かい感情に寄り添います。

---

## 機能

- **9つの感情タグ**から今の気分を選ぶだけ
- ヨルシカ・amazarashi・BUMP OF CHICKEN・RADWIMPS・サカナクション・藤井風・宇多田ヒカル・アニメOSTなど**手動キュレーション**の全34曲
- 「もう一曲」で同じ気分の別の曲へ（全曲消費したら一周通知）
- YouTubeで聴くボタン付き
- **シェア機能**（Web Share API対応・非対応時はクリップボードコピー）
- AIなし・外部API不要・完全ローカル動作

---

## 感情タグ一覧

| カテゴリ | タグ |
|---------|------|
| 落ち込み | もの悲しい / 胸が痛い / なんか泣きたい |
| 前向き | 前に進みたい / 背中を押してほしい |
| モヤモヤ | 頭を空っぽにしたい |
| 穏やか | ただ落ち着きたい / 懐かしい気持ち |
| テンション | 最高にアガりたい |

## ローカルで起動

```bash
pnpm install
pnpm dev
```

ブラウザで `http://localhost:5173` を開く。

---

## 曲を追加したい場合

`src/data/songs.ts` の `SONGS` 配列にオブジェクトを追加します。

```ts
{
  id: 'ユニークなID',
  title: '曲名',
  artist: 'アーティスト名',
  reason: 'この感情にこの曲な理由（一言）',
  youtubeUrl: 'https://www.youtube.com/results?search_query=...',
  tags: ['対応するタグID'], // 複数可
}
```

タグIDは `TAGS` 配列の `id` フィールドを参照してください。
